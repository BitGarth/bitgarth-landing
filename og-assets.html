BitGarth — social assets

Two cards rendered at exact pixel dimensions. To export each as a PNG:

  1. Open this file in Chrome (or any Chromium-based browser).
  2. Open DevTools (⌥⌘I on macOS, F12 on Windows/Linux).
  3. In the Elements panel, right-click the <section class="card-frame og"> node → Capture node screenshot. Saves a pixel-perfect 1200×630 PNG.
  4. Repeat for <section class="card-frame x"> to get 1500×500.
  5. Rename the files og-card.png and x-banner.png, drop them into assets/.

Tip: turn the safe-zone overlays on while composing, off before capturing.

Section iii is a live preview: drop, click, or paste any candidate banner image onto it to see how it sits with the real avatar and the mobile crop bands. No upload — the image stays in your browser.
